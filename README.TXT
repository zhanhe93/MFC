
【安装方式】

1. 在硬盘中建立一个子目录（名称自取）
2. 将光盘片内容自行拷贝到硬盘子目录中（可利用 xcopy）
3. 利用 Windows 95 资源管理器（explorer），管理或执行各范例程序

例如，您的光驱代码为 E:，您希望安装到 C:\Dissect 目录中。可以在
DOS BOX 内做以下动作：

> C:
> CD \
> MD Dissect
> CD Dissect
> xcopy E:\Dissect\*.* .
